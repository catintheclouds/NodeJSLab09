Step 1: Install NodeJS
Step 2: Download all files
Step 3: Run Terminal / Command Prompt and go to the folder with the downloaded files
Step 4: On Terminal / Command Prompt, type "npm init" (without the ")
Step 5: Press enter many times and type "Yes" at the end when prompted.
Step 6: Check that a folder "node_modules" has been made.
Step 7: Run lab09.js by typing "node lab09.js" (without the ")
